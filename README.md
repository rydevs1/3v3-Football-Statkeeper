# 3v3-Football-Statkeeper
Football Statkeeper is built and used in a small winter/summer 3v3 football league with personalized rules (for the smaller 3v3 game). The main purpose is to log game results live and in-person with an easy-to-use button system, and then have the ability to saved finished game logs and box score stats at the end of the game.
